Instruções da Avaliação

Obrigado por participar da Avaliação do nosso projeto ElementAR. Esse documento irá lhe guiar durante o processo.

História

Você é Bit, um robô que se perdeu viajando entre os mundos e ficou preso no nosso. Imbuído com os poderes dos elementos, Bit deve utilizar portais para voltar a seu mundo.

Como Jogar

ElementAR é um jogo que usa realidade aumentada, assim, você irá utilizar a câmera do celular para controlar Bit e outros elementos do jogo.

Bit

É o seu personagem. Bit tem a habilidade de alterar o seu elemento, podendo alterar entre fogo, eletricidade, água, pedra, ar e gelo.

Para controlar o Bit, você precisa montar conexões entre os comandos, até o portal de saída. Quando uma conexão válida for criada, você poderá comandar Bit a seguir o caminho.

Comandos

São objetos que alteram o estado de Bit. Na versão atual, esses são dois, os comandos Condicionais e os Laços. Comandos podem ser ligados a outros, a Bit e ao portal de saída ao serem pressionados. Isso irá abrir o menu, com opções para Conectar e Desconectar.

Ao selecionar a opções de Conectar, você será instruído a pressionar outro objeto, criando uma conexão entre os dois. Para desconectar, o fluxo é o mesmo, o que irá remover a conexão, caso exista.

Condicionais

Condicionais são comandos que recebem o elemento atual de Bit e realizam uma checagem para determinar com qual elemento ele será imbuído a seguir. Caso o elemento de Bit combine com o elemento a ser checado, seu elemento irá mudar para o elemento verdadeiro, indicado pelo anel verde. Caso contrário, irá mudar para o elemento falso, indicado pelo anel vermelho.

Laços

Laços não alteram o elemento de Bit diretamente, mas eles podem ser conectados com outros condicionais para que Bit possa repetir o mesmo caminho um determinado número de vezes. Um laço não pode conter Bit, o portal de saída e nem outros laços.

Existem dois tipos de conexões relacionadas aos laços, as conexões tradicionais, identificadas pela cor verde, e as suas conexões internas, identificadas pela cor vermelha. 

Fases

O jogo é composto por 5 fases. Em cada fase, seu objetivo é fazer Bit chegar ao portal com um elemento específico utilizando comandos e conexões, alterando seu estado de elemento. Em cada fase, o nível de complexidade irá subir, e você precisará montar caminhos maiores para fazer Bit chegar ao objetivo.
